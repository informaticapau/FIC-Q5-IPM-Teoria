# Flutter y Dart

## Flutter

- Boilerplate
- Todo son _widgets_
- Favorece la composición sobre la herencia
- Favorece el estilo declarativo

La vista es una función del estado

## Dart

Funciones anónimas: `(parámetros){sentencias;}`

Azúcar sintáctico:

```dart
(x, y) {return z;}
(x, y) => z
```

Métodos privados: Empiezan por "_".
